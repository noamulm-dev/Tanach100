
הוראות התקנה ודיפלוי ידני (Manual Deployment)
=============================================

כדי להעלות את האפליקציה לשרת באופן ידני, עקוב אחר השלבים הבאים:

1. יצירת קובץ המידע (Bundle):
   --------------------------
   א. פתח את האפליקציה בדפדפן (בסביבת הפיתוח).
   ב. כנס לתפריט הראשי -> לחץ על מספר הגרסה (בתחתית התפריט, באדום) 7 פעמים ברצף.
   ג. במסך המפתח שנפתח, לחץ על "יצירת קובץ הפצה (Bundle)".
   ד. המתן לסיום ושמור את הקובץ בשם `tanakh_bundle.json`.

2. הכנת מבנה הקבצים לשרת:
   ----------------------
   לאחר שבנית את האפליקציה (npm run build), עליך לוודא שבתיקיית ההפצה (dist/build) קיים המבנה הבא.
   שים לב: התיקייה `data` צריכה להיות באותה רמה כמו `index.html`.

   / (שורש האתר)
   ├── index.html          (קובץ האפליקציה)
   ├── assets/             (קבצי הקוד והעיצוב)
   └── data/               (תיקייה חדשה שתיצור ידנית)
       └── books/          (תיקייה בתוך data)
           └── tanakh_bundle.json   <--- הקובץ שהורדת בשלב 1

3. בדיקה:
   -------
   לאחר ההעלאה, נסה לגלוש לכתובת הישירה של הקובץ כדי לוודא שהוא נגיש:
   https://your-website.com/data/books/tanakh_bundle.json
   
   אם הקובץ יורד או מוצג בדפדפן, האפליקציה תצליח לטעון אותו, גם אם האפליקציה מותקנת בתוך תיקייה פנימית (Sub-directory).
